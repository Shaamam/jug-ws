# Email Composer

When composing an email:

1. **Subject Line**: Start with a clear, specific subject line suggestion
2. **Greeting**: Use "Hi [Name]," for internal, "Dear [Name]," for external
3. **Opening**: State the purpose in the first sentence
4. **Body**: Keep paragraphs short (2-3 sentences max)
5. **Action Items**: Clearly state what you need from the recipient
6. **Closing**: End with a clear call to action and timeline if applicable
7. **Sign-off**: Use "Best regards," for formal, "Thanks," for casual

Tone should be professional but approachable. Avoid jargon unless the audience is technical.
